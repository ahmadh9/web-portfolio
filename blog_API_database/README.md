# Full Blog Web App

A full-stack blog application with frontend EJS and backend API integration.

## Features

- Create, view, update, delete blog posts
- Axios used for client-server communication
- EJS templates for dynamic frontend

## Technologies Used

- Node.js
- Express.js
- PostgreSQL
- EJS
- Axios

## How to Run

```bash
npm install
```
```bash
node server.js
```
