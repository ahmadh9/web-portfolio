# Web Development Portfolio – Ahmad Hammad

This repository contains a curated collection of my full-stack and frontend web development projects completed during my MERN Stack Bootcamp and personal learning journey.

Each folder showcases a separate project with its own structure and README. Projects demonstrate my skills in JavaScript, Node.js, Express.js, PostgreSQL, API integration, responsive design, and DOM manipulation.

## 📂 Projects Overview

- **Full Blog Web App**  
  A complete blog system using Node.js, Express.js, EJS, and PostgreSQL with full CRUD operations and structured codebase.

- **Blog API (Best Practice)**  
  Backend API architecture demonstrating clean Express.js structure with PostgreSQL.

- **User Management System**  
  A Vanilla JavaScript app using OOP and DOM to manage user input, validation, and live preview in cards/tables.

- **NASA Homepage Clone**  
  A static responsive replica of NASA's official homepage built using HTML, CSS, and Bootstrap.

- **WhatsApp Web Clone**  
  A minimal UI clone using only HTML and Flexbox, optimized for responsiveness.

## 🔧 Technologies Used

- Frontend: HTML, CSS, JavaScript, Bootstrap, EJS
- Backend: Node.js, Express.js
- Database: PostgreSQL
- API Handling: REST APIs, Axios
- Tools: Git, GitHub, VS Code

## 📎 Notes

Each project is contained in its own folder and includes a `README.md` with setup instructions.  
Projects are actively improved and reflect my current learning track as a MERN developer.
